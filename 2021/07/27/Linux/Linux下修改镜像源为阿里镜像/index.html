Linux下修改镜像源为阿里镜像

基于CentOS Linux release 7.8.2003

1、首先安装weget

yum install wget

2、备份你的原镜像文件，避免出错后可以恢复(这里是把原文件重新命名)

cd /etc/yum.repos.d/
mv CentOS-Base.repo CentOS-Base.repo.bak
3、下载新的CentOS-Base文件
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

4、运行yum makecache生成缓存

yum makecache
